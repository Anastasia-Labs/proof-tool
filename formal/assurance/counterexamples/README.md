# Counterexample evidence

This directory is reserved for minimized, replayable counterexamples produced
by the formal theorem catalog.

A saved counterexample must identify the theorem/catalog ID, exact compiled
artifact hashes, Lean/Blaster/Z3 versions, ledger-validity result, serialized
V3 context, and independent compiled-contract replay. Model-only or malformed
non-ledger witnesses must be labeled as such and are not on-chain findings.

Do not store seed phrases, master keys, derivation paths, private proving
material, wallet files, provider credentials, or transaction-signing material.
